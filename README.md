# Stadium-mate
⚽ StadiumMate: An intelligent, multilingual chatbot using the Gemini API to provide fans with real-time navigation, accessibility, and transport info at World Cup 2026 stadiums.


About The Project
StadiumMate is a GenAI-enabled web application designed to enhance the stadium experience for fans attending the FIFA World Cup 2026. Leveraging the Gemini API and a mini-RAG (Retrieval-Augmented Generation) pattern, it acts as a smart, multilingual assistant. Fans can ask questions in their native language about gates, food, accessible routes, and transportation, and StadiumMate instantly provides accurate answers grounded in real stadium data.
